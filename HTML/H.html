Problem H: Hogemon Get

Problem

がっちょ君は今、人気のHogemon Getというゲームに熱中している。
がっちょ君が住んでいる会津国は1からNの番号がついているN個の町からなり、 町と町とは道で結ばれている。会津国にはM本の道があり、すべての道は異なる2つの町を結んでいる。がっちょ君は道を双方向に自由に移動できるが、道以外を通って、ある町から別の町に行くことはできない。

Hogemon Getでは、町iでボールをdi個入手することができる。
しかし、ある町で再びボールを入手するためには、最後にその町でボールを入手してから15分以上経過している必要がある。

がっちょ君は最初、町1にいて、町NにR分以内で移動しようと考えている。
そこで、がっちょ君は移動の際に、できるだけたくさんのボールを集めたい。
最大でいくつのボールを入手することができるだろうか。

Input

入力は以下の形式で与えられる。

N M R
d1 d2 ... dN
a1 b1 c1
a2 b2 c2
...
aM bM cM

入力はすべて整数である。
1行目は町の個数N,道の本数M,制限時間Rが空白区切りで与えられる。
2行目には町i(i=1,2,...,N)に訪れることで入手することができるボールの個数diが空白区切りで与えられる。
3行目からM+2行目には道j(j=1,2,...,M)の情報aj,bj,cjが空白区切りで与えられる。j番目の道は町ajと町bjの間をcj分で移動できることを表す。

Constraints

入力は以下の条件を満たす。

  • 3 ≤ N ≤ 30
  • N-1 ≤ M ≤ min(N×(N-1)/2, 300)
  • 10 ≤ R ≤ 1000
  • 0 ≤ di ≤ 10
  • d1 = dN = 0
  • 1 ≤ aj < bj ≤ N
  • 5 ≤ cj ≤ 100
  • 町1から町NへはR分以内で移動できることが保証されている
  • ある2つの町の組に対して2本以上の道があることはない

Output

町1から町NへR分以内に移動するまでに入手することができる最大のボールの個数を1行で出力せよ。

Sample Input 1

5 4 40
0 1 1 1 0
1 2 5
2 3 5
3 4 5
4 5 5

Sample Output 1

6
