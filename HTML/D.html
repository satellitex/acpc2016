Problem D: Magical Girl Sun Ranger

Story

魔法少女サンレンジャーは、悪の組織から人々を守るために日々活動をしている。
彼女は敵を倒すときの決め技「テラ粒子レクイエムシュート」を持っていて、それは、両手でハート型を作りながらステッキを持ち、ステッキから攻撃を繰り出すというものである。

このステッキは内部で魔力をチャージし、発射するという仕組みになっている。
内部には、複数の魔力を増幅できる点と1つの魔力を使い決め技を発射する点がある。
さらに魔力を伝達させるための装置があり、これらは、それぞれ、2つの点を結び、すべての魔力をまとめて伝達させることができる。
しかし、伝達できる魔力の量が定まっており、その量を下回ったり、超えて伝達しようとすると、制御が効かなくなり、ステッキが壊れてしまうため、そのようなことはできない。

最初、ある点に魔力1を生成する。これ以降、新たに魔力の生成することができない。
サンレンジャーは、発射点に魔力が伝達される際に、できるだけ魔力を大きくしたいと考えている。

魔力は妖精が運び、妖精は、プログラムによって制御できる。
サンレンジャーに依頼されたあなたは、プログラムを書いて発射点に伝達することが可能な魔力の最大値を調べることにした。

Problem

番号1~NのN個の点とM個の線がある。
最初、妖精は点1にいて、値が1の数を持っている。
そして、最終的に点Nへ行きたいと思っている。

点1~点N-1は今持っている数に1以上の任意の値を加算させられる点である。
また、i番目の線は、点aiからと点biへ移動するできる。
ただし、この線を通るためには、持っている数の値がちょうどciでなければならない。

妖精が点Nへ到達したときの数の値を最大化せよ。もし、点Nへ到達できない場合は、-1を出力せよ。

Input

入力は以下の形式で与えられる。

N M
a1 b1 c1
a2 b2 c2
…
aM bM cM

1行目には、2つの整数N, Mが空白区切りで与えられる。 2行目からM+1行目のそれぞれの行iには、3つの整数ai, bi, ciが空白区切りで与えられる。

Constraints

入力は以下の条件を満たす。

  • 2 ≤ N ≤ 105
  • 1 ≤ M ≤ 2 × 105
  • 1 ≤ ai < N
  • 1 ≤ bi ≤ N
  • 1 ≤ ci ≤ 109
  • ai と bi は相異なる。

Output

点Nへ到達できる場合は、その最大の魔力を1行に出力する。できない場合は、変わりに-1を出力する。

Sample Input 1

3 2
1 2 10
2 3 20

Sample Output 1

20

まず、点1で魔力を10まで上げ、点2へ魔力を伝達させる。
次に、点2で魔力を20まで上げ、点3へ魔力を伝達させる。
以上より、点3に魔力20で伝達させることができる。

Sample Input 2

4 4
1 2 27
1 3 37
2 3 47
3 1 57

点4への伝達装置がないため、点4へ魔力を伝達させることができない。

Sample Output 2

-1

Sample Input 3

3 3
1 2 13
2 3 17
2 3 15

Sample Output 3

17

まず、点1で魔力を13まで上げ、点2へ魔力を伝達させる。
次に、点2からの伝達装置には、点3へ魔力17で伝達できるものと魔力15で伝達できるものがあるので、より大きい魔力17で伝達させる。
以上より、点3に魔力17で伝達させることができる。
