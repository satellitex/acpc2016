Problem D: Magical Girl Sun Ranger

Story

魔法少女サンレンジャーは、悪の組織から人々を守るために日々活動をしている。
彼女は敵を倒すときの決め技「テラ粒子レクイエムシュート」を持っていて、それは、両手でハート型を作りながらステッキを持ち、ステッキから攻撃を繰り出すというものである。

このステッキは内部で魔力をチャージし、発射するという仕組みになっている。
内部には、魔力を増幅できる複数の点と魔力を使い決め技を発射する1つの点がある。
さらに魔力を転送させるための装置があり、これらは、それぞれ、2つの点を結び、すべての魔力をまとめて転送させることができる。
しかし、転送できる魔力の量が定まっており、その量を下回ったり、超えて転送しようとすると、制御が効かなくなり、ステッキが壊れてしまうため、そのようなことはできない。

最初、ある点に魔力量1を生成する。これ以降、新たに魔力を生成することができない。
サンレンジャーは、発射点に魔力が転送される際に、できるだけ魔力量を大きくしたいと考えている。

魔力は妖精が運び、妖精は、プログラムによって制御できる。
サンレンジャーに依頼されたあなたは、プログラムを書いて発射点に転送することが可能な魔力量の最大値を調べることにした。

Problem

番号1からNまでのN個の点とM個の転送装置がある。
最初、妖精は点1にいて、魔力量1を持っている。
そして、最終的にできるだけ多くの魔力を持って、点Nへ行きたいと思っている。

1からN-1までの点ではその時点で持っている魔力量に1以上の任意の魔力量を加算することができる。
また、i番目の転送装置は、魔力を点aiから点biへ移動させることができる。
ただし、この転送装置を使うためには、持っている魔力量がちょうどciでなければならない。

妖精が点Nへ到達したときの魔力量の最大値を出力せよ。もし、点Nへ到達できない場合は、-1を出力せよ。

Input

入力は以下の形式で与えられる。

N M
a1 b1 c1
a2 b2 c2
…
aM bM cM

1行目には、2つの整数N, Mが空白区切りで与えられる。
2行目からM+1行目のそれぞれの行iには、3つの整数ai, bi, ciが空白区切りで与えられる。

Constraints

入力は以下の条件を満たす。

  • 2 ≤ N ≤ 105
  • 1 ≤ M ≤ 2 × 105
  • 1 ≤ ai < N
  • 1 ≤ bi ≤ N
  • 1 ≤ ci ≤ 109
  • ai ≠ bi は相異なる

Output

点Nへ到達できる場合は、その最大の魔力量を1行に出力する。到達できない場合は、代わりに-1を出力する。

Sample Input 1

3 2
1 2 10
2 3 20

Sample Output 1

20

まず、点1で魔力を10まで上げ、点2へ魔力を伝達させる。
次に、点2で魔力を20まで上げ、点3へ魔力を伝達させる。
以上より、点3に魔力20で伝達させることができる。

Sample Input 2

4 4
1 2 27
1 3 37
2 3 47
3 1 57

点4への伝達装置がないため、点4へ魔力を伝達させることができない。

Sample Output 2

-1

Sample Input 3

3 3
1 2 13
2 3 17
2 3 15

Sample Output 3

17

まず、点1で魔力を13まで上げ、点2へ魔力を伝達させる。
次に、点2からの伝達装置には、点3へ魔力17で伝達できるものと魔力15で伝達できるものがあるので、より大きい魔力17で伝達させる。
以上より、点3に魔力17で伝達させることができる。
