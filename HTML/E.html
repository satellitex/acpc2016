Problem E: Graph making

Problem

どのノードとも連結でないn個のノードがある。
各ノード間に長さ1の無向辺を張っていく。
直径(各ノードと連結のノードとの最短距離のうち最大)がdになるようにした時、最大何本辺を張ることができるか求めよ。
多重辺、自己ループは認めない。

Input

n d

入力は全て整数で与えられる。
1行にnとdが空白区切りで与えられる。

Constraints

入力は以下の条件を満たす。

  • 2 ≤ n ≤ 109
  • 1 ≤ d ≤ n−1

Output

張ることができる最大の辺の数を1行に出力せよ。

Sample Input 1

4 3

Sample Output 1

3

Sample Input 2

5 1

Sample Output 2

10

Sample Input 3

4 2

Sample Output 3

5
