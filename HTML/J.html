Problem J: Char Swap

Problem

長さが偶数の文字列Sが与えられる。
あなたは、隣り合った2つの文字を入れ替える操作を何回でも行うことができる。
文字列を回文にするためには、最小で何回の操作を行う必要があるだろうか?
不可能な場合は-1を出力せよ。

Input

入力は以下の形式で与えられる。

S

1行に文字列Sが与えられる。

Constraints

入力は以下の条件を満たす。

  • 2 ≤ |S| ≤ 400000
  • 文字列はすべてアルファベットの小文字で構成されている。

Output

文字列Sを回文にするための最小の操作回数を1行に出力する。不可能な場合は変わりに-1を出力をする。

Sample Input 1

acca

Sample Output 1

0

Sample Input 2

acpcacpc

Sample Output 2

3

Sample Input 3

aizu

Sample Output 3

-1
