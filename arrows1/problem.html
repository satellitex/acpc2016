Problem C: Credit

Problem

月曜日から金曜日の1限からN限までの時間割表があり、M個の履修可能な科目がある。
科目iは、曜日di (di = 0が月曜日, di = 1が火曜日, ...,di = 4が金曜日)のai限から始まり、連続してkiコマで行われ、その単位数はtiである。

科目を重ならないようにL個まで選んだときに取れる単位数を最大化せよ。
なお、受講した講義は必ず単位が取れるものとする。

Input

入力は以下の形式で与えられる。

N M L
d1 a1 k1 t1
d2 a2 k2 t2
...
dN aN kN tN

1行目に3つの整数N, M, Lが空白区切りで与えられる。
2行目からM+1行目に4つの整数di, ai, ki, tiが空白区切りで与えられる。

Constraints

入力は以下の条件を満たす。

  • 2 ≤ N ≤ 300
  • 0 ≤ M ≤ 300
  • 0 ≤ L ≤ M
  • 0 ≤ di ≤ 4
  • 1 ≤ ai ≤ N
  • 1 ≤ ki
  • ai + ki - 1 ≤ N
  • 1 ≤ ti ≤ 100

Output

科目を重ならないようにL個まで選んだときに取れる単位数を最大化する。

Sample Input 1

3 7 3
0 1 1 1
0 1 1 2
1 1 3 4
1 1 1 1
1 2 1 2
2 1 1 3
2 2 2 1

Sample Output 1

9

Sample Input 2

5 10 5
0 1 1 2
0 2 1 2
0 1 2 3
1 2 1 2
1 4 2 3
2 1 1 1
2 1 1 2
3 3 2 3
4 1 1 2
4 2 1 2

Sample Output 2

13
